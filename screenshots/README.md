# Screenshots

This folder contains dashboard screenshots for the README.

## Adding Screenshots

1. Export your dashboard screenshot from Grafana (Dashboard → Share → Snapshot or use browser screenshot)
2. Anonymize any sensitive data (IPs, hostnames, etc.)
3. Save as PNG with the dashboard name (e.g., `network-overview.png`)
4. Recommended size: 1200x800px or similar 3:2 ratio

## Required Files

- `network-overview.png`
- `suricata-ids.png`
- `dns-analytics.png`
- `opnsense-firewall.png`
- `system-health.png`
- `wireguard-vpn.png`
